# AI 작업 지침

이 파일의 규칙은 이 저장소에서 작업하는 모든 AI 도구와 세션에 적용한다.

## 필수 문서 확인

- 작업을 시작하기 전에 `PLAN.md`를 읽는다.
- 첫 화면 또는 사건 카드를 수정할 때는 `DESIGN_GUIDE.md`를 반드시 읽고 승인된 디자인을 유지한다.
- 이미지를 추가하거나 교체할 때는 `images/README.md`의 출처 기록 형식을 따른다.

## Git 커밋 규칙

- 사용자가 커밋을 요청하면 일반 개발자 커밋처럼 작성한다.
- 커밋 메시지 제목과 본문에 AI로 작업했다는 표현을 넣지 않는다.
- `Co-authored-by`, `Co-worked-by`, `Generated-by` 등 AI 도구 또는 AI 공동작업자를 나타내는 트레일러를 절대 추가하지 않는다.
- `Codex`, `Claude`, `ChatGPT`, `AI` 등의 이름을 커밋 메시지에 자동으로 넣지 않는다.
- 커밋 작성자 또는 공동 작성자를 AI 이름으로 변경하지 않는다.
- 별도 요청이 없으면 간결하고 작업 내용을 직접 설명하는 커밋 메시지를 사용한다.

## 현재 승인된 첫 화면 원칙

- 사건 카드는 시작 시점이 오래된 순서로 정렬한다.
- 전쟁 카드에는 실제 대표 사진과 실제 세력·전선 지도 두 장을 사용한다.
- 위에는 전쟁 사진, 아래에는 그레이스케일 지도를 배치하고 그라데이션으로 합성한다.
- 국가·대륙·한반도 윤곽을 임의로 그려 지도처럼 사용하지 않는다.
- 외부 자료는 `images/`에 저장하고 출처를 `images/README.md`에 기록한다.
- 세부 CSS 값과 전쟁별 이미지 조합은 `DESIGN_GUIDE.md`를 기준으로 한다.
